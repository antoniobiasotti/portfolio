3 formas de usar a função print() em Python

A função print é sem dúvida uma das mais básicas.

January 19, 2025

# 1º Método - Concatenação de Strings
nome = "Antônio"
idade = 22
print("Meu nome é " + nome + " e tenho" + idade + " anos.")

# 2º Método - .format()
nome = "Antônio"
idade = 22
print("Meu nome é {} e {} anos.".format(nome, idade))
